# v25 패치 노트 - 전체화면 버튼 / 우측 BGM 미니 슬라이더

- 인트로 포함 전체 게임 화면 좌측 하단에 `전체화면` 버튼을 추가했습니다.
- 좌측 하단 `전체화면` 버튼에 마우스를 올리면 반투명 선택 박스와 글로우가 표시됩니다.
- 버튼 클릭 시 전체화면 전환, 전체화면 상태에서 다시 클릭 시 전체화면 해제가 동작합니다.
- 게임 타이틀 화면과 본 게임 진행 화면에서 MENU 버튼 위에 BGM 볼륨 아이콘을 추가했습니다.
- BGM 볼륨 아이콘에 마우스를 올리면 볼륨 조정 바가 펼쳐지고, 벗어나면 아이콘만 보이게 수정했습니다.
- 펼쳐진 BGM 볼륨 조정 바는 마우스 클릭/드래그로 직접 조절할 수 있습니다.
- BGM 볼륨 조정 시 음소거 상태는 자동 해제됩니다.
- 기존 설정 화면의 볼륨 조정 기능과 키 설정 기능은 그대로 유지됩니다.
