# v44 - 내 동꼽즈 기록 최근 게임 기록 레이아웃 수정

## 수정 내용
- `내 동꼽즈 기록` 화면의 `최근 게임 기록` 마지막 줄이 박스 밖으로 벗어나는 문제 수정
- 최근 기록 행 간격과 폰트 크기를 조정하여 7줄이 박스 안에 깔끔하게 들어가도록 수정
- 기록 행 표시 영역에 클리핑 처리를 추가하여 스크롤/행 수 증가 시에도 박스 밖으로 튀어나오지 않게 보정
- 가독성을 위해 짝수 행에 은은한 반투명 배경을 추가
