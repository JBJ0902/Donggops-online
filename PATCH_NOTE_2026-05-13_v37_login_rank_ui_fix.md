# v37 로그인/랭킹 UI 및 효과음 음소거 개선

## 변경 사항
- 타이틀 화면의 `Google 로그인` 표기를 `로그인`으로 간소화했습니다.
- 로그인 완료 안내 문구에서 `Google` 표기를 제거하고 `로그인 완료` 중심으로 수정했습니다.
- 로그인 드롭다운의 `G-xxxxxx` 플레이어 코드를 숨겼습니다.
- 내 기록 페이지에서 내부 `google:...` ID 대신 닉네임/로그인 상태/랭킹 연동 상태만 표시하도록 수정했습니다.
- 동꼽즈 게임 모드 화면의 사용자 경쟁전 아래에 `사용자 온라인 경쟁전 순위 보기` 메뉴를 추가했습니다.
- 설정 메뉴에 `효과음 음소거` 버튼을 추가하고, 게이지 바와 겹치지 않도록 배치를 조정했습니다.
- 경쟁전 패배 화면의 다음 안내를 `ENTER : 초기 화면`으로 변경했습니다.
- 사용자 경쟁전 랭킹 화면 제목을 `사용자 경쟁전 순위`로 변경했습니다.
- 외부 랭킹 연동 문구를 `외부 경쟁전 랭킹 : 연동 완료`로 변경했습니다.
- 랭킹 동기화 중 문구를 반복 출력하지 않고, 필요 시 `외부 경쟁전 순위` / `갱신 완료`만 짧게 표시하도록 변경했습니다.
- 사용자 경쟁전 순위 화면 우측 상단에 `순위 갱신` 버튼을 추가했습니다.
- `순위 갱신` 버튼은 마우스 오버 시 반투명 박스가 표시되고, 클릭 시 외부 랭킹을 다시 불러옵니다.
- 경쟁전 위험 경고 문구는 `위험!`으로 유지되며, 1~10스테이지 전체에서 작동합니다.
- Apps Script 설정 저장에 `sfx_muted` 컬럼을 추가하여 효과음 음소거 상태도 저장 가능하도록 보강했습니다.
