# v47 동꼽 해방 사용 방식 및 관리자 테스트 모드 수정

## 적용 내용

- 7~10스테이지의 AI 상승폭은 변경하지 않았습니다.
- 7~10스테이지 동꼽 자동사냥 사용 방식을 수정했습니다.
  - 기존: MAX일 때 자동사냥 키 1회 입력 시 동전 전부 즉시 사용
  - 변경: 자동사냥 키 1회 입력마다 동전 1개씩 차감
  - MAX 상태에서 시작해 동전을 전부 사용하면 동꼽 해방 발동
- 동꼽 해방 발동 시 10초간 동꼽 키를 누르고 있는 동안만 제한 자동연타가 작동합니다.
- 동꼽 해방 중이 아닐 때는 동꼽 키/별풍선 키를 누르고 있어도 자동연타 편법이 작동하지 않습니다.
- 7~10스테이지 동전 1개당 13초 CPM 보정 유지
  - 7스테이지: +450 CPM
  - 8스테이지: +500 CPM
  - 9스테이지: +550 CPM
  - 10스테이지: +600 CPM
- 1~6스테이지 기존 밸런스는 유지했습니다.
- 관리자 전용 히든 테스트 모드 보강
  - Ctrl + Shift + T: 기존 관리자 밸런스 테스트
  - Ctrl + Shift + Tab: 동꼽 해방 테스트 모드
  - 동꼽 해방 테스트 모드에서는 7~10스테이지 시작 시 MAX 코인이 자동 지급됩니다.
  - 해방이 끝나면 다시 MAX 코인이 자동 충전되어 반복 테스트가 가능합니다.
  - 테스트 모드는 기록/랭킹 저장이 되지 않습니다.
