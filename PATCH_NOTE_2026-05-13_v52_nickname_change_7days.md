# PATCH NOTE v52 - 로그인 후 닉네임 변경 메뉴 추가

## 수정 내용
- 타이틀 화면 우측 상단 로그인 계정 메뉴에 `닉네임 변경` 버튼을 추가했습니다.
- 닉네임 변경은 7일에 1번만 가능하도록 제한했습니다.
- 변경 가능 시간이 남아 있으면 남은 시간 안내가 표시됩니다.
- Google 로그인 사용자는 닉네임 변경 시 Apps Script / Google Sheets에도 동기화됩니다.
- 기존 경쟁전 랭킹 기록의 닉네임도 변경된 닉네임으로 갱신되도록 Apps Script를 보강했습니다.
- 로그인 시 기존 사용자의 닉네임을 클라이언트 값으로 임의 덮어쓰지 않도록 변경했습니다.
