# PATCH NOTE v23 - 별풍선/자동사냥 사용 가능 VFX 안내 강화

## 수정 내용
- 게임 HUD 상단 조작 안내를 3구역으로 분리하여 긴 문구가 서로 겹치지 않게 정리했습니다.
- 별풍선 키가 활성화된 동안 `별풍선 터짐(키 활성)` 문구가 다이나믹하게 색상 변화되도록 수정했습니다.
- 별풍선 활성 문구 뒤에 가벼운 반투명 VFX 박스, 스캔라인, 별빛 이펙트, `USE!` 안내를 추가했습니다.
- `동꼽(자동사냥) x5/5` 상태일 때 색상 변화, 반투명 VFX 박스, 스캔라인, 별빛 이펙트, `READY!` 안내가 표시되도록 수정했습니다.
- VFX는 Canvas 내부 고정형 효과로 구현하여 파티클 증가 없이 렉이 걸리지 않도록 제한했습니다.

## 업로드 방법
- GitHub에는 `game.js` 파일을 덮어쓰기 업로드하면 됩니다.
- `assets` 폴더는 이번 패치에서 변경 사항이 없습니다.
