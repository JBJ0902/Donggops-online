# PATCH NOTE v22 - 인트로 클릭 UI / 볼륨 접힘 / AI 대전 카운트다운

## 수정 사항
- 인트로 좌측 하단 `1/6` 페이지 표시 삭제
- 인트로 진행 문구를 `CLICK : 다음`으로 변경
- `CLICK : 다음` 버튼에 마우스를 올리면 반투명 선택 박스가 보이고 클릭 시 다음 인트로로 진행
- 인트로 진행 버튼을 하단 스토리 자막과 겹치지 않도록 위쪽으로 재배치
- 인트로 우측 볼륨 UI는 기본 상태에서 아이콘만 표시하고, 마우스를 올릴 때만 슬라이더가 펼쳐지도록 수정
- 싱글 플레이 AI 대전 1~5스테이지 시작 전 `게임시작 3초전 → 2초전 → 1초전 → 게임 시작!` 카운트다운 추가
- 카운트다운 중에는 스테이지 BGM이 먼저 재생되며, 카운트다운 종료 후 실제 플레이 시간이 시작됨
