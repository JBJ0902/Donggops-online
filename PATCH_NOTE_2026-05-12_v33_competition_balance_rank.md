# v33 사용자 경쟁전 밸런스 패치

- 사용자 경쟁전도 스테이지 시작 전 3초 카운트다운 적용
- 경쟁전 1~5스테이지에서 AI가 500점 이상 앞서면 빨간 경고 알람 VFX 표시
- AI가 1000점 이상 앞서면 즉시 패배 처리 후 경쟁전 결과 화면으로 이동
- 경쟁전 6~10스테이지 자동사냥 최대 보유량 확장: 6/7/8/9/10개
- 사용자 경쟁전 최종 결과 화면의 `도달` 표기를 `스테이지`로 변경

참고: GitHub Pages 단독 정적 호스팅은 서버 DB 쓰기 기능이 없으므로, 외부 PC 간 공용 랭킹 저장은 Firebase/Supabase/별도 API 같은 백엔드가 필요합니다. 현재 랭킹은 브라우저 localStorage 기반입니다.
