# PATCH NOTE v42 - 사용자 경쟁전 패배 후 순위 화면 이동

## 수정 내용
- 사용자 경쟁전 패배 화면의 안내 문구를 `ENTER : 초기 화면`에서 `ENTER : 사용자 경쟁전 순위 보기`로 변경했습니다.
- 사용자 경쟁전 패배 후 ENTER/클릭 시 초기 화면이 아니라 사용자 경쟁전 순위 화면으로 이동하도록 수정했습니다.
- 10스테이지 클리어처럼 경쟁전이 종료되는 경우에도 순위 화면으로 자연스럽게 이동하도록 흐름을 정리했습니다.
- 순위 화면 진입 시 외부 경쟁전 순위 갱신을 자동으로 시도하도록 유지했습니다.
