# v13 온라인 로비 / READY / 카운트다운 / 채팅 패치

## 추가/수정 사항
- 온라인 1:1 대전을 즉시 시작 방식에서 온라인 로비 방식으로 변경했습니다.
- 온라인 로비에서 닉네임을 입력할 수 있습니다.
  - 한글 포함 닉네임: 최대 12자
  - 영문 닉네임: 최대 18자
- 방장과 참가자가 모두 READY를 눌러야 게임이 시작됩니다.
- READY 완료 후 3초 카운트다운을 추가했습니다.
  - 게임 시작 3초전
  - 게임 시작 2초전
  - 게임 시작 1초전
  - 게임 시작!
- 온라인 대전 플레이 시간을 방장이 선택할 수 있게 했습니다.
  - 2분 / 3분 / 4분 / 5분
- 온라인 대전 BGM을 방장이 선택할 수 있게 했습니다.
  - 시작화면 BGM
  - 캐릭터 선택 BGM
  - 스테이지 1~5 BGM
- 온라인 대전은 1판 종료 후 온라인 로비로 복귀하도록 변경했습니다.
- ESC 또는 우측 하단 MENU를 누르면 온라인 경기 중 로비로 돌아가도록 변경했습니다.
- 상대가 로비로 나가면 남은 플레이어도 로비로 돌아가도록 처리했습니다.
- WebRTC 연결이 끊어지면 안내 후 모드 선택 화면으로 복귀하도록 처리했습니다.
- 온라인 로비에 간단한 채팅 기능을 추가했습니다.
- 기존 싱글 AI 대전 기능은 유지했습니다.
