# PATCH NOTE 2026-05-13 v40 keyconfig_pause_fix

## 적용 내용

- 기본 동꼽 자동사냥 키가 새로 시작한 게임에서 바로 동작하지 않던 문제 수정
  - Google Sheets/Apps Script 또는 localStorage에서 숫자 키가 number 타입으로 저장되어도 문자열 키 입력과 정상 매칭되도록 정규화
  - 기본값: 동꼽 SPACE / 별풍선 B / 동꼽 자동사냥 5 유지

- 키 설정 화면 개선
  - `키 설정 초기화` 버튼 추가
  - 클릭 시 동꼽 SPACE, 별풍선 B, 동꼽 자동사냥 5로 즉시 복구
  - 로그인 사용자 설정에도 저장되도록 처리

- MENU / ESC로 설정 화면 진입 시 일시정지 화면 뒤쪽 캐릭터 연타 애니메이션이 계속 움직이던 문제 수정
  - 설정 화면이 열린 동안 플레이어/AI 캐릭터 프레임 표시를 정지
  - 메뉴를 닫으면 기존처럼 정상 애니메이션 재개
