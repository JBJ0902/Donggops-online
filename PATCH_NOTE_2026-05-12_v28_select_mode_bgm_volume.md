# v28 선택/모드 화면 BGM 볼륨 아이콘 패치

- 캐릭터 선택 화면에서도 우측 하단 MENU 버튼과 전체화면 아이콘 사이에 BGM 볼륨 아이콘이 표시되도록 수정했습니다.
- 동꼽즈 게임 모드 화면에서도 동일하게 BGM 볼륨 아이콘이 표시되도록 수정했습니다.
- BGM 볼륨 아이콘에 마우스를 올리면 반투명 패널과 볼륨 조정 바가 펼쳐집니다.
- 펼쳐진 볼륨 바를 마우스로 클릭/드래그하여 BGM 볼륨을 조정할 수 있습니다.
- 볼륨 조정 시 BGM 음소거가 자동 해제되도록 유지했습니다.
- MENU, 전체화면, BGM 볼륨 아이콘이 서로 겹치지 않도록 기존 우측 하단 버튼 배치를 유지했습니다.
