# v45 관리자 스테이지 점프 테스트 모드

## 적용 내용
- 관리자 전용 히든 밸런스 테스트 모드 추가
- 모드 선택 화면에서 `Ctrl + Shift + T` 입력 시 관리자 테스트 화면 진입
- 닉네임이 `GM`, `ADMIN`, `JBJ`, `관리자`인 로그인 계정만 접근 가능
- 1~10스테이지를 버튼 클릭 또는 숫자키로 바로 시작 가능
  - `1~9`: 해당 스테이지
  - `0`: 10스테이지
- 테스트 모드는 사용자 경쟁전 AI 밸런스를 그대로 사용하되 기록/랭킹에는 저장하지 않음
- 테스트 종료 후 ENTER/CLICK 시 관리자 스테이지 선택 화면으로 복귀
- 게임 중 상단 표기를 `관리자 테스트`로 표시하여 일반 경쟁전과 구분

## 사용 방법
1. GM 닉네임 계정으로 로그인
2. 캐릭터 선택 후 동꼽즈 게임 모드 화면으로 이동
3. `Ctrl + Shift + T` 입력
4. 원하는 스테이지 클릭 또는 숫자키 입력
