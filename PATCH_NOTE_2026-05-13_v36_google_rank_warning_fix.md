# v36 Google 계정 표시 / 경쟁전 경고 / 외부 랭킹 동기화 보강

## 수정 내용
- Google 로그인 후 화면에 노출되던 긴 `google:...` 내부 ID를 숨김 처리
- 계정 표시를 `Google 연동 · 플레이어 코드 G-xxxxxx` 형태로 변경
- 내 기록 페이지에 동기화 상태 표시 추가
- 사용자 경쟁전 경고 문구를 `삐용 삐용!`에서 `위험!`으로 변경
- 사용자 경쟁전 탈락 위험/탈락 판정을 1~10스테이지 전체에 적용
- 사용자 경쟁전 최종 결과 페이지에서 Google Sheets 외부 랭킹 동기화 상태 표시
- 최종 결과 진입 시 Google Sheets 랭킹을 강제 새로고침하도록 보강

## 업로드
- 기존 GitHub 파일에 덮어쓰기 업로드
- 최소 업로드: `game.js`, 이 패치노트
- 전체 업로드: 압축 해제 후 전체 파일 덮어쓰기
