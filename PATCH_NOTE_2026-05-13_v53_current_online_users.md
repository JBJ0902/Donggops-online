# PATCH NOTE v53 - 현재접속자 표시 기능 추가

## 수정 내용
- 로그인 후 우측 상단 계정 메뉴에 `현재접속자 : X명` 항목을 추가했습니다.
- 개인정보 안내문에 현재접속자 표시를 위한 `접속 상태` 저장 항목을 반영했습니다.
- Google 로그인 성공 시 Apps Script 서버에 접속 상태를 등록합니다.
- 로그인 상태에서는 약 45초마다 접속 상태를 갱신합니다.
- 로그아웃 또는 브라우저 종료 시 접속 종료 요청을 보내도록 처리했습니다.
- Apps Script에 `online_sessions` 시트를 추가했습니다.
- 서버에서는 최근 2분 이내에 갱신된 사용자만 현재접속자로 계산합니다.
- 현재접속자 항목을 클릭하면 즉시 새로고침됩니다.

## 적용 위치
- game.js
- apps_script/Code.gs
- GOOGLE_SHEETS_LOGIN_SETUP.md
