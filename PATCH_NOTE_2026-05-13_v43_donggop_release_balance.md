# PATCH NOTE v43 - 7~10스테이지 동꼽 해방 밸런스

## 적용 내용
- 7~10스테이지 AI 상승폭은 변경하지 않음.
- 사용자 경쟁전 7스테이지부터 `동꼽 해방` 기능 추가.
- 7~10스테이지에서 동꼽 자동사냥 적립치가 MAX일 때 자동사냥 키를 누르면 보유 동전을 전부 사용.
- MAX 사용 시 10초 동안 동꼽 키를 누르고 있으면 제한된 자동연타 보조가 발동.
- MAX 사용 시 13초 동안 동전 1개당 스테이지별 CPM 보정 적용.
  - 7스테이지: 동전 1개당 +450 CPM
  - 8스테이지: 동전 1개당 +500 CPM
  - 9스테이지: 동전 1개당 +550 CPM
  - 10스테이지: 동전 1개당 +600 CPM
- 기존 1~6스테이지 동꼽 자동사냥 밸런스는 유지.
- 동꼽 해방 중 keydown 반복 이벤트를 직접 점수로 반영하지 않고, 게임 루프 안에서 제한된 자동연타로 처리하여 꾹누르기 편법 재발 방지.
- HUD에서 7~10스테이지는 `동꼽 해방 x현재/MAX`로 표시하고, 해방 지속시간과 총 CPM 보정이 보이도록 개선.
