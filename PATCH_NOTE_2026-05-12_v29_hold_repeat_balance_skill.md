# PATCH NOTE 2026-05-12 v29 - 꾹누르기 자동연타 방지 / 밸런스 스킬화

## 적용 내용
- 동꼽 키와 별풍선 키를 누르고 있을 때 브라우저 키 반복 입력으로 자동연타가 발생하던 편법을 차단했습니다.
- 같은 키를 누르고 있는 동안에는 첫 입력 1회만 인정하고, 키를 떼었다가 다시 눌러야 다음 입력으로 인정되도록 변경했습니다.
- 자동연타성 효과는 편법이 아니라 아래 조건형 기술로만 발동되도록 정리했습니다.
  - 별풍 리액션 러시: 순수 키보드 연타 660CPM 이상을 5초 유지하면 10초간 자동 +500CPM 발동
  - 동꼽 자동사냥: 순수 키보드 연타 1000CPM 이상을 3초 유지하면 획득, 사용 시 13초간 자동 +300CPM 발동
- 설정 > 게임 설명 문구도 위 규칙에 맞게 수정했습니다.
- 키를 길게 누르고 있을 때는 “꾹누르기 무효! 순수 연타만 인정” 안내가 과도하게 반복되지 않도록 제한했습니다.

## 밸런스 의도
- 기본 점수는 순수 연타 실력으로 확보합니다.
- 일정 CPM을 달성한 플레이어만 시간제한 자동 보조 기술을 얻을 수 있게 하여 난이도와 보상감을 유지했습니다.
- 파티클/VFX 추가 없이 입력 처리만 최적화하여 렉 부담을 늘리지 않았습니다.
