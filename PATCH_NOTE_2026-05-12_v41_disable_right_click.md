# v41 우클릭 메뉴 차단 패치

## 적용 내용
- 전체 게임 페이지에서 마우스 우클릭 기본 메뉴 비활성화
- 캔버스, 온라인 접속 패널, 로그인 패널 등 게임 내 UI 전체에 적용
- 마우스 중간 클릭/보조 클릭의 기본 동작도 함께 차단
- PC/모바일 브라우저에서 이미지 저장 메뉴가 쉽게 뜨지 않도록 `user-select`, `touch-callout` 차단 CSS 추가
