# 동꼽즈 AI & ONLINE GitHub Pages 버전

이 버전은 GitHub Pages 업로드용 경량 브라우저 게임입니다.

## 포함 기능

- 싱글 플레이 AI 대전
- WebRTC P2P ONLINE 1:1 대전
- 시작 화면 BGM
- 캐릭터 선택/모드 선택 BGM
- 1~5스테이지 각각 다른 BGM
- 1스테이지 2분 / 2스테이지 2분 30초 / 3스테이지 3분 / 4스테이지 3분 30초 / 5스테이지 5분
- 동전 효과음
- 아이템 준비/사용 효과음
- 점수/피버/승리 효과음
- 슈니/다시바 3단 동전 꼽기 모션
- 별풍선 B키 10초 해금
- D키 기능 제거
- 동꼽 아이템
- ESC 또는 우측 하단 MENU로 설정/일시정지
- 별풍선 CPM 충전 바 표시
- AI 400점 이상 차이 시 추격 보정

## 로컬 테스트

1. 압축 해제
2. `LOCAL_TEST_START.bat` 실행
3. 브라우저에서 접속

```text
http://127.0.0.1:8000
```

## GitHub Pages 업로드

GitHub에 zip 파일을 올리지 마세요.  
압축을 푼 뒤 아래 파일/폴더만 올리세요.

```text
index.html
style.css
game.js
assets/
README_설치방법.md
UPLOAD_CHECKLIST.md
.gitignore
.nojekyll
```

업로드 후:

```text
Settings → Pages
Source: Deploy from a branch
Branch: main
Folder: /root
Save
```

주소 예:

```text
https://본인아이디.github.io/Donggops-online/
```

## 조작법

```text
ENTER / SPACE : 진행
A / ← : 슈니 선택
B / → : 다시바 선택
ENTER / SPACE : 싱글 플레이 AI 대전
O : ONLINE P2P 대전
SPACE : 동꼽
B : 별풍선 발동 시 7초 해금
D : 4스테이지부터 조건 달성 시 7초 해금
5 : 동꼽(자동사냥) 사용
```

## ONLINE P2P 연결

방장:
1. O 키
2. 방 만들기
3. 생성된 방 코드를 상대에게 전달
4. 상대 답변 코드를 받아 붙여넣기

참가자:
1. O 키
2. 방 참가
3. 방장이 준 코드를 붙여넣기
4. 생성된 답변 코드를 방장에게 전달

## 주의

WebRTC P2P는 VPN, 회사/학교 방화벽, UDP 차단, 일부 LTE/5G symmetric NAT 환경에서는 연결이 실패할 수 있습니다.
