# v15 온라인 로비 UI / 표시 패치

- 온라인 로비에서 MENU 버튼이 로비 박스와 겹치지 않도록 오른쪽 여백으로 이동했습니다.
- 온라인 게임 진행 중 MENU 버튼 표기를 LOBBY로 변경했습니다.
- 온라인 로비 복귀 안내 문구가 BGM 선택 박스와 겹치지 않도록 상단 알림 영역으로 이동했습니다.
- 로비 채팅 메시지, 입력창, 전송 버튼이 서로 겹치지 않도록 채팅 박스 배치를 조정했습니다.
- 긴 입력 문구는 입력창 안에서만 보이도록 클리핑 처리했습니다.
- 온라인 대전 점수판에서 PLAYER/P2 아래에 닉네임을 표시하고, 그 아래 캐릭터 이름을 별도로 표시하도록 변경했습니다.
- 온라인 로비의 준비/나가기 버튼과 플레이 시간/BGM 옵션 박스의 배치를 정리했습니다.
